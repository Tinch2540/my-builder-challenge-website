Linux Filesystem Management Project

In this project, I created and managed a multi-level company directory structure in Linux to simulate real-world enterprise departments. The work focused on using essential Linux commands such as mkdir -p, cd, ls -R, and touch to build, navigate, and manage directories and files efficiently. The project demonstrates not only technical execution but also problem-solving skills, such as handling permission issues with sudo and correcting errors caused by typos or missing directories.

I structured a virtual company with departments such as Finance, IT, and Marketing, each containing relevant subfolders and files (e.g., budget.xlsx, app.py, survey.csv).

Key Highlights

  • Used mkdir -p to recursively create directories at multiple levels in one command.
  • Navigated the filesystem using cd and verified the hierarchy with ls -R.
  • Created departmental files with touch, adding realism to the structure.
  • Applied sudo to manage permission restrictions.
  • Debugged path issues (e.g., corrected a typo: Developments → Development).

Skills Demonstrated

  • mkdir -p for recursive directory creation
  • cd and ls -R for navigation and structure verification
  • touch for file creation
  • sudo for overcoming permission issues
  • Debugging path errors and correcting directory naming mistakes
